GET /api/hello¶

A test function returning hello world

Query Parameters:
 
  • name (string) – a name to display
Response JSON Array of Objects:
 
  • name (string) – a name to display, not present when error occurs
  • date (string) – today’s date, not present when error occurs
  • message (string) – error message, present when error occurs
  • action (string) – how to correct error, present when error occurs
Status Codes:
  • 200 OK – returns a token of name and date
  • 400 Bad Request – returns an error token
